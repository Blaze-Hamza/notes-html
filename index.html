HTML NOTES

All the HTML Elements/Tags used in building web pages

FORM

A form is an HTML element that collects input from users — like names, emails, messages, or any other data — and sends it to a server for processing.

Why forms are important:

  • log in pages
  • sign up pages
  • contact forms
  • search bars

Form Elements:

  • <form> - Container for the form element
  • <input> - User input field
  • <textarea> - Long text input
  • <select> - Dropdown menu
  • <button> - Clickable button

TABLE

An HTML table is a structured way to display data in rows and columns on a web page. It helps organize information for easy reading and comparison.

Why tables are important:

  • Data organization
  • Comparison
  • Clarity
  • Professionalism

Table Elements:

  • <table> - Container for the table
  • <tr> - Table row
  • <th> - Table header cell
  • <td> - Table data cell
Example of a table:
Name Age City
Hamza 18 Nairobi
Hailey 25 Cape Town
Sujal 19 Mumbai


LINKS

An HTML link is a clickable element on a web page that allows users to navigate to another page or resource when clicked. Links are created using the <a> (anchor) tag in HTML.

Why links are important:

  • Navigation
  • Connectivity
  • SEO
  • User Experience
Example of a link:

Visit Example.com



HEADINGS

HTML headings are used to define the structure and hierarchy of content on a web page. They range from H1 (most important) to H6 (least important).

Why headings are important:

  • Content Organization
  • SEO Benefits
  • User Experience
  • Accessibility
Example of Headings:

This is Heading 1

This is Heading 2

This is Heading 3

This is Heading 4

This is Heading 5
This is Heading 6


IMAGES

An HTML image is a visual element that is displayed on a web page using the <img> tag. Images enhance the visual appeal and provide context to the content.

Why images are important:

  • Visual Appeal
  • Context
  • User Engagement
