FORM

A form is an HTML element that collects input from users — like names, emails, messages, or any other data — and sends it to a server for processing.

Why forms are important:

  • log in pages
  • sign up pages
  • contact forms
  • search bars

Form Elements:

  • <form> - Container for the form element
  • <input> - User input field
  • <textarea> - Long text input
  • <select> - Dropdown menu
  • <button> - Clickable button
